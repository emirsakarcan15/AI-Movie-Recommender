# 🎬 WhatToWatch

An AI-powered movie recommendation web application that suggests films based on the user's mood, preferences, or natural language prompts.

Users can simply type something like:

> “A dark sci-fi movie for a rainy night”

and receive intelligent movie recommendations generated with AI and enriched using movie data from TMDB.

---

# 🚀 Features

- 🤖 AI-powered movie recommendations using Gemini API
- 🎥 Movie metadata and posters from TMDB API
- ⚡ Fast and responsive UI
- 🎨 Modern minimalist interface
- 🔍 Natural language movie search
- 📱 Responsive design
- 🌙 Smooth animations and transitions
- 🔐 Environment variable management with `.env`
- 🔄 CI/CD pipeline with GitHub Actions

# 🧠 How It Works

```text
User Prompt
   ↓
React Frontend
   ↓
Express Backend
   ↓
Gemini AI analyzes the prompt
   ↓
Recommended movie titles
   ↓
TMDB API fetches posters & metadata
   ↓
Results displayed to the user
```
